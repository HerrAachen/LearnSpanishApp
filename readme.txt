Learn Spanish App

Android App for practicing and displaying the conjugation of Spanish Verbs.
Includes:
 - LearnSpanishApp: The actual Android App
 - LearnSpanishAppTest: contains the test cases
 - LearnSpanishAppPreparation: A project which creates the database needed for the app
 
The verbs and their conjugations are extracted from the Spanish Wiktionary (http://es.wiktionary.org/).
